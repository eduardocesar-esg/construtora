# PRM-BOOTSTRAP-001 - Codex Repository Setup

## Objetivo

Configurar ou revisar a estrutura governada do repositorio do piloto PMO Agentico para construtora de medio porte.

## Entrada Esperada

- Nome do projeto.
- Escopo do piloto.
- Ciclos planejados.
- Lista de artefatos minimos.
- Regras de governanca.

## Tarefa do Codex

1. Verificar se a estrutura minima do repositorio existe.
2. Criar arquivos ausentes sem sobrescrever conteudo existente relevante.
3. Garantir que registros CSV tenham colunas de governanca.
4. Garantir que prompts, agentes, templates, dashboard e politicas tenham owner, versao e status.
5. Sinalizar lacunas de informacao para revisao humana.

## Saida Esperada

- Estrutura de pastas criada ou validada.
- Lista de arquivos criados ou ajustados.
- Pontos pendentes para decisao do PMO.

## Regras

- Nao automatizar decisoes criticas.
- Nao remover conteudo existente sem autorizacao.
- Manter rastreabilidade entre artefatos, prompts, agentes e evidencias.
